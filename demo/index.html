What?

collectorium provides pure CSS tabs which are completely functional without JavaScript.

Why?

First and foremost I just really like doing things, which are usually done with JavaScript, without JavaScript. Next my focus was to keep the file size of the CSS file (which provides the core functionallity) at an absolute minimum. So it is possible to provide a basic tab functionallity for minimalistic sites with a focus on loading time and mobile performance.

I think the core functionality of websites should be accessible without JavaScript: read more about why progressive enhancement is important.

How?

The short answer is: I use the CSS :target pseudo class. If you want to go into detail, I tried my best to document the code with comments.
